Disable the annoying shutdown tracker pop-up on Multi-session hosts

Problem:
When publishing multi session hosts on Citrix infrastructure using Windows server as host OS, We will be coming across an inconvienient feature that will be showing for all the users after publishing new features or as part of maintenance that will involove power-cycling the VM. as shown below:
Solution:
To disable this pop-up, we can use the following:

  • Make sure the host computer objects on the AD are located under a separate OU
  • Open Group policy management editor
  • Navigate to Computer Configuration > Policies > Administrative Templates > System > Display Shutdown Tracker and set it to disabled as shown below:
